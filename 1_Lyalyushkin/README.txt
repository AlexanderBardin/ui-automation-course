Текст задания:
    Написать Python скрипт, который выводит список запущенных приложений и их главных окон.
    Для описания приложения достаточно bundle_id и pid.
    Для описания окна нужен title и состояние (свёрнуто, максимизировано).
    Разрешается использовать только ctypes и встроенные средства Python на OS X 10.5+.
    Нельзя использовать pyatom (atomac) и нежелательно pyobjc (см. ответ с ctypes примером на StackOverflow).
    В случае использования pyobjc реализовать также функции или методы start/kill для приложения.

Использование:
    python AppList.py

Формат вывода:
    bid: pid
        title: number: isHidden
